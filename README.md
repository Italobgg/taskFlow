## ✨ Funcionalidades

- ➕ Criar, editar e excluir tarefas
- ✅ Marcar tarefas como concluídas
- 🏷️ Categorizar tarefas por tipo
- 🔍 Filtrar tarefas por status e categoria
- 💾 Persistência de dados com LocalStorage
- 📱 Design responsivo (mobile-first)

## 🧪 Testes

O projeto possui cobertura de testes utilizando **Vitest** e **React Testing Library**.

### O que é testado

| Área | Descrição |
|------|-----------|
| Componentes | Renderização, interações e estados |
| Hooks | Lógica de CRUD, filtros e persistência |
| Integração | Fluxo completo de gerenciamento |

### Comandos de Teste

| Comando | Descrição |
|---------|-----------|
| npm run test | Executa os testes em modo watch |
| npm run test:ui | Interface visual dos testes |
| npm run coverage | Relatório de cobertura |

## 🚀 Como Executar

### Pré-requisitos

- Node.js v18 ou superior
- npm ou yarn

### Instalação

Clone o repositório:

git clone https://github.com/Italobgg/taskFlow.git
cd taskFlow

Instale as dependências:

npm install

Execute o projeto:

npm run dev

Acesse: http://localhost:5173
